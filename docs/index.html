makeSafe

This function helps you to clear the inputs/gets value from. It can be used with int values like :

$PHPultimate->makeSafe($_GET['id'],true,true);

or you can use it to clear the input value from HTML code like this :

$PHPultimate->makeSafe($_POST['message'],false,true);
Parameter Type Required Default value Description
txt string Yes null The text you want to clear
intval boolean No False True if you want to return the integer value.
no_html boolean No false True if you want to clear the string from HTML tags.

createRandomKey

This functions generates random letters/numbers or both of them. You can also choose how many characters you want to display.

To generate 10 random numbers : 

echo $PHPultimate->createRandomKey(10, 1);

Output :

3161973823

To generate 10 random letters : 

echo $PHPultimate->createRandomKey(10, 'a');

Output :

vqmPvRvaFo

To generate 10 different characters : 

echo $PHPultimate->createRandomKey(10, 'all');
echo $PHPultimate->createRandomKey(10);

Output :

ROS9KW8Zbb
Parameter Type Required Default value Description
amount integer NO 5 The amount of characters you need.
char string NO all The type of characters you want. 1 for numbers, a for letter, all for both.

redirect

This function is designed to redirect users to any page you want in a given number of seconds.

to redirect to http://google.com immediatly use the following coce :

echo $PHPultimate->redirect("http://google.com");

to redirect to http://google.com in 10 seconds usee :

echo $PHPultimate->redirect("http://google.com", 10);
Parameter Type Required Default value Description
url string Yes null The url you want to redirect to.
seconds integer NO 0 Number of seconds you want to redirect after.

counter

This function works the same as the function above expect that with this function you can show a message and a counter before redirecting.

echo $PHPultimate->counter("http://google.com", "You'll be sent to google in x seconds.", 10);

Output :

You'll be sent to google in 10 seconds.

Note : After the counter hits 0, You'll be redirected to http://google.com.

Parameter Type Required Default value Description
url string Yes null The url you want to redirect to.
text string Yes null The text you want to show with the counter.
seconds integer NO 0 Number of seconds you want to redirect after.

embedYoutube

Embed any YouTube video with this function and customize the embed box as you want.

echo $PHPultimate->embedYoutube("Kdgt1ZHkvnM", 500, 240, 'light', 0, 0);

Output :

To make the video starts automatically use this :

echo $PHPultimate->embedYoutube("Kdgt1ZHkvnM", 500, 240, 'light', 1, 0);

For the dark theme of the embed :

echo $PHPultimate->embedYoutube("Kdgt1ZHkvnM", 500, 240, 'dark', 0, 0);

To display the video information :

echo $PHPultimate->embedYoutube("Kdgt1ZHkvnM", 500, 240, 'light', 0, 1);
Parameter Type Required Default value Description
url string Yes null The YouTube video url or ID.
width integer NO 640 The box width.
height integer NO 390 The box height.
theme string NO dark The embed theme it can be "light" or "dark".
autoplay integer NO 0 autoplay the video, 1 for Yes, 0 for No.
showinfo integer NO 0 show the video information, 1 for Yes, 0 for No.
